*help.txt*	Pour Vim version 7.0. Dernière modification : 7 mai 2006

			   VIM - Sommaire de l'aide

        Déplacements : Utilisez les flèches du clavier, ou bien	  }     k
		       "k" pour haut, "j" pour bas,		  }   h   l
		       "h" pour gauche, "l" pour droite.	  }     j
Fermer cette fenêtre : Tapez ":q<Entree>".
       Sortir de Vim : Tapez ":qa!<Entree>" (attention, toutes les
		       modifications seront perdues !).

Sauter vers un sujet : Placez le curseur sur un marqueur (par exemple
                       |barres|) et tapez CTRL-]
                       [N.D.T. : Ctrl + AltGr + ')' sur un clavier français].
      Avec la souris : Tapez ":set mouse=a" pour rendre la souris active (dans
		       un xterm ou dans l'interface graphique).
		       Double-cliquez sur un marqueur, par exemple |barres|.
  Revenir en arrière : Tapez CTRL-T ou CTRL-O (plusieurs retour en arrière
                       sont possibles).

Pour une aide ciblée : Il est possible d'obtenir directement de l'aide sur
		       n'importe quel sujet en donnant un argument à la
		       commande d'aide |:help|.
		       On peut également préciser un contexte d'utilisation :
							*help-context*
		       CONTEXTE                     PRÉFIXE   EXEMPLE     ~
		       commande en mode Normal      (aucun)   :help x
		       commande en mode Visuel         v_     :help v_u
		       commande en mode Insertion      i_     :help i_<Esc>
		       commande en mode Lignecmd       :      :help :quit
		       édition en mode Lignecmd        c_     :help c_<Del>
		       argument de la commande Vim     -      :help -r
		       option                          '      :help 'textwidth'
Rechercher de l'aide : Tapez ":help chaîne", puis CTRL-D pour lister toutes
		       les entrées de l'aide contenant « chaîne ».

VIM est l'acronyme de "Vi IMproved" (VI aMélioré). Ce logiciel a été réalisé
en grande partie par Bram Moolenaar, mais l'aide de nombreux contributeurs
s'est révélée déterminante. Voir |credits|.

==============================================================================
*bars* *barres*	    Illustration pour les barres

Si vous avez sauté jusqu'ici avec CTRL-] ou un double clic de la souris, vous
pouvez utiliser CTRL-T, CTRL-O, g<ClicDroit>, ou <C-ClicDroit> pour retourner
où vous étiez.

Notez que les marqueurs sont entre des caractères « | », mais lorsque la
coloration syntaxique est activées, ils sont cachés, ce qui rend le contenu
plus facile à lire.

De plus, vous pouvez utiliser CTRL-] sur n'importe quel mot, même s'il n'est
pas entre deux « | », et Vim essayera de trouver l'aide correspondante.

 vim:tw=78:fo=tcq2:isk=!-~,^*,^\|,^\":ts=8:ft=help:norl:
